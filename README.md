This is a quiz application created with a React frontend and an asp.net web api backend and has verification. This project connects to a MSSQL Server.

To run this application, start the backend api within visual studio. Move to the quiz-client folder and run:

<code>npm install</code>

Then to start up the front-end run:

<code>npm start</code>
